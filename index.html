🇬🇧 Beginner's Walkthrough:
  1. CF API Token: Click Api Token Template ↗. Click Use Template (if prompted), then scroll down and click Continue to summary > Create Token. Copy that token and paste it above!
  2. CF Account ID: Open any Cloudflare Workers or Domain page in your browser. Look at the URL bar: copy the 32-character string between dash.cloudflare.com/ and the next slash (e.g. dash.cloudflare.com/YOUR_ACCOUNT_ID/workers).
  3. Worker Script Name: Go to Compute > Workers & Pages in Cloudflare. Copy your worker's name (e.g. nahan or nahan-gateway).
🇮🇷 راهنمای مبتدیان فارسی:
  1. توکن دسترسی کاربری CF API Token: روی لینک میانبر قالب توکن ↗ کلیک کنید. در پایین صفحه دکمه Continue to summary و سپس Create Token را بزنید و توکن نمایش داده شده را کپی کنید.
  2. شناسه اکانت ابری CF Account ID: کافیست وارد یکی از صفحات کلودفلر خود شوید. از آدرس بالای مرورگر، عبارت ۳۲ کاراکتری که بلافاصله بعد از dash.cloudflare.com/ قرار دارد را بردارید (مثلاً: dash.cloudflare.com/a1b2c3d4e5...).
  3. نام کارگر (Worker Name): از منوی کناری کلودفلر به بخش Compute > Workers & Pages رفته و نام ورکر خود را عینا بنویسید (مثلاً nahan).
